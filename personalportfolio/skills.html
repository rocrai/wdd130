HTML

HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page. HTML consists of a series of elements. HTML elements tell the browser how to display the content.

CSS

CSS is a stylesheet language used to describe the presentation of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

Python

Python is an interpreted, object-oriented, high-level programming language with dynamic semantics developed by Guido van Rossum.
